# Каскада Маркет

Простой интернет-магазин для покупателей.

## Что внутри

- главная витрина;
- большой баннер;
- категории;
- кружочки-истории;
- поиск;
- акционные баннеры;
- карточки товаров;
- нижнее мобильное меню.

## Запуск локально

```bash
npm install
npm run dev
```
